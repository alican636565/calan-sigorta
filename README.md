# Calan Sigorta — Next.js (Vercel Ready)

## Hızlı Başlangıç
```bash
npm i
npm run dev
```
Lokal: http://localhost:3000

## Yayınlama (Vercel)
1. Bu klasörü GitHub’a yükle.
2. vercel.com → New Project → GitHub reposunu bağla → Deploy.
3. Özel alan adı: Vercel > Domains.
